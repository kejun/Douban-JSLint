Douban-JSLint
------------------------------------------------------------------------------
是针对“Douban Javascript Core Style Guideline”(http://bit.ly/douban-javascript)对JSLint的规则做了调整。更符合开发的实际需求。

安装
------------------------------------------------------------------------------
首先需要安装NodeJS和npm。
npm install http://github.com/kejun/Douban-JSLint/tarball/master

使用
------------------------------------------------------------------------------
douban-jslint *.js
douban-jslint file1.js file2.js
